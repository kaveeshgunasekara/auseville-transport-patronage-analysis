# AusVille Transport Patronage Analysis

## Overview
This project analyzes 8 years of Melbourne public transport data to understand COVID-19's impact on patronage patterns and guide service planning decisions.

## Dataset
Monthly average patronage data (2018-2025) across 5 transport modes - MetroBus, MetroTrain, RegionalBus, RegionalTrain, and Tram. Dataset includes 4,300 observations with day type, day of week, and passenger counts.

## Tech Stack
Python | pandas | NumPy | SciPy | Matplotlib | Seaborn | Power BI | DAX | Jupyter Notebook

## Key Findings
* 85% demand collapse in March 2020 (COVID-19 lockdowns)
* Uneven recovery: Regional services 95-99% recovered, metro services only 67%
* Mode explains 70% of variance - strongest demand predictor
* 98% weekday premium - extreme commuter dependency
* 265,000 daily passengers permanently lost from metro services

## Analysis Components
* Data cleaning and feature engineering (Period categories, rolling averages)
* Statistical testing (Welch's t-tests, ANOVA, effect sizes)
* 3 page interactive Power BI dashboard with 16 DAX measures
* Segmented comparison modeling (Pre-2020 / COVID / Post-2021)

## Deliverables
* Interactive Power BI dashboard (.pbix)
* Python analysis notebook (.ipynb)
* Academic report (PDF)
* Clean dataset (CSV)

## Outcome
Recommended 15% cost optimization through metro frequency reduction, regional service expansion strategy, and infrastructure deferral guidance based on permanent demand shifts.
